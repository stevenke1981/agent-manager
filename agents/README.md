## 使用方式

### 在 VS Code Copilot Chat 中使用
1. 將整個 gents/ 目錄複製到你的工作目錄
2. 在 Copilot Chat 中輸入 @workspace 後選擇對應的 Agent
3. 或直接在提示中說「請扮演 [agent名稱] 說明 [主題]」

### 在 AgentSkills.io 中使用
按照平台說明匯入各個 SKILL.md 檔案。

### 在 Claude Code 中使用
把 agents 目錄放置在專案根目錄，Claude 會自動識別 SKILL.md 格式。

## Agent 格式規範

每個 SKILL.md 包含：
```yaml
---
name: agent-name
description: 一行說明，含觸發情境
metadata:
  author: luckyegg168
  version: "1.0"
  category: 分類名稱
  language: zh-TW
---

# Agent 主體內容（Markdown）
```

## 倫理聲明

- **爭議行業類別**（14-17類）均採用「研究/分析師」視角，以識別、預防和反制為目標
- 所有 Agent 均不提供違法活動的操作性指導
- 成人娛樂業類別（15類）限18歲以上使用
- 本資料庫的 Agent 用於學習、研究、創作和職業模擬
